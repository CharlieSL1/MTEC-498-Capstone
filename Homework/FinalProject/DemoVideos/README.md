# Demo Videos Directory

This directory should contain demo videos (2-3 minutes each) demonstrating key functionality of the PanGu system.

## Required Demo Videos

### 1. Key Functionality Walkthrough (2-3 minutes)
**Content**:
- System overview and setup
- Depth camera tracking demonstration
- Gesture recognition showcase
- OSC communication explanation
- Max for Live integration
- Spatial audio control demonstration

**Format**: MP4, 1080p minimum, with clear audio narration or captions

### 2. Live Performance Demonstration (2-3 minutes)
**Content**:
- Live performance setup
- Real-time gesture-controlled spatial audio
- Multiple gesture types in action
- Spatial positioning demonstration
- Audience perspective (if available)

**Format**: MP4, 1080p minimum, with clear audio

### 3. Technical Implementation Overview (Optional, 2-3 minutes)
**Content**:
- Code walkthrough (if appropriate)
- Architecture explanation
- Technical challenges and solutions
- Development process highlights

**Format**: MP4, 1080p minimum

## Video Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: Minimum 1080p (1920x1080)
- **Frame Rate**: 30fps minimum
- **Audio**: Clear narration or captions
- **File Size**: Optimized for web/portfolio use
- **Duration**: 2-3 minutes per video

## File Naming Convention

Use descriptive filenames:
- `functionality_walkthrough.mp4`
- `live_performance_demo.mp4`
- `technical_overview.mp4`

## Upload Locations

After creating videos, consider uploading to:
- YouTube (for easy sharing)
- Google Drive (for portfolio)
- Vimeo (for professional presentation)

Include links in the main FinalProject README.md
